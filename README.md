# 网站底层项目

此项目为网站最底层的代码，只实现系统权限管理部份：

1、用户管理

2、角色管理

3、菜单管理

4、自动生成系统菜单

5、为角色授权菜单

6、为用户分配角色

基本就只实现这些功能，在开发其他项目时可直接将该项目复制一份即可扩展。
